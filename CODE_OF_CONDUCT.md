# Contributor Covenant Code of Conduct

## Our Pledge

We pledge to make participation in this project respectful, inclusive, and constructive for contributors of all backgrounds and experience levels.

## Expected Behavior

- Be respectful when reviewing code, results, assumptions, and methods.
- Critique ideas and evidence, not people.
- Prefer clear reproduction steps over blame.
- Respect data privacy, security boundaries, and attribution requirements.

## Unacceptable Behavior

- Harassment, intimidation, or discriminatory language.
- Publishing private data, credentials, or sensitive operational details.
- Misrepresenting model results, benchmarks, or scientific claims.
- Personal attacks in issues, pull requests, or project discussions.

## Enforcement

Maintainers may remove comments, reject contributions, or restrict participation when behavior conflicts with this code of conduct. Security-sensitive incidents should be reported using `SECURITY.md`.
